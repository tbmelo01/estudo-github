Como Clonar um Repositório:

Comando:

git clone (url)

Exemplo:

git clone https://github.com/tbmelo01/estudo-github.git

Como salvar suas alterações nos arquivos:

1.passo: selecionar os aquivos que você deseja observar

git add (nome do aquivo)

git add index.html

git add .

2 passo: criar o ponto de restauração

git commit -m "(mensagem)"

git commit -m "criado os arquivos iniciais"

Como subir as alterações para o github

Depois de ter realizado o commit, faça essa etapa

git push

Como atualizar a pasta com as alterações do github

Depois de ter realizado o commit, faça essa etapa

git pull
